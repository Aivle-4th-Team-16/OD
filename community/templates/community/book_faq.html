결제 후, 유료 콘텐츠 이용 내역이 없으시면 환불 가능합니다. (결제 후 7일 이내 환불 요청시 적용됩니다)

가입하신 계정 확인을 위해 1:1 문의로 환불 요청해주시면 확인 후 안내해드리겠습니다.

*환불 불가의 경우

1.구독 후 이용내역이 있는 경우

(가족 공유 서비스 이용 시, 초대받으신 분이 이용내역이 있으실 경우 환불이 불가합니다)

2.쿠폰 구매 후, 등록하신 경우

*단, 아이폰 결제의 경우 앱 스토어 정책으로 앱 스토어에서 환불을 진행해주셔야합니다.

여러 기기(디바이스)에서 같은 아이디로 오디오북을 이용하실 수 있어요 :D

다만, 동시 이용은 불가능합니다.

메인 화면에서 원하는 오디오북을 선택 -> 현재 선택 성우 사진을 클릭해 성우 선택 -> 페이지 가운데 청취하기 버튼 클릭 -> 재생 버튼 클릭

저희 서비스는 원하는 성우를 커스텀으로 제작 가능해요.

10분 이상의 음성 파일을 준비해주세요.

성우 선택 -> 개인 탭 -> + 버튼 클릭 -> 음성파일 업로드 및 이름 지정 -> 생성하기(약 1시간 소요) -> 저장하기

이후 개인 성우 내 음성에서 녹음한 성우를 확인할 수 있어요.

다른 사람의 성우를 원할경우 인기 탭을 이용해주세요.

이번주 HOT 음성은 사용자분들의 청취 수에 의해 결정돼요.

매주 월요일에 갱신되며, 실시간으로 반영됩니다.

오른쪽 상단 프로필 클릭 -> 내 정보 -> 좋아요 내역을 통해 확인하실수 있어요.

신청된 책은 매주 갱신되어 반영돼요. 결과는 이메일로 보내드리고 있어요.
